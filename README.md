# jupyter notebook 화면 설정 방법  
C:\사용자\\admin\\.jupyter 폴더에 쥬피터 노트북 설정이 있다.  
그 아래에 custom 이라는 이름의 폴더를 만들고, 그 아래에 custom.css 라는 파일을 복사해 넣는다.

.jupyter 폴더가 보이지 않으면 까만 화면에서 jupyter notebook --generate-config를 실행한다.
